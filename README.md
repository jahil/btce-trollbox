BTC-E Trollbox
==============
btc-e.com trollbox viewer for console lovers ;)


Installation
------------
	git clone https://github.com/alanmcintyre/btce-api.git
	cd btce-api 
	sudo python setup.py install
	sudo apt-get install redis-server
	sudo pip install colorama


Usage
-----
	python trollbox.py
